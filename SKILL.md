# 毛泽东选集 · Mao Zedong Selected Works

A structured knowledge base extracted from the 1,258-page, 5-volume compendium of Mao Zedong's key writings (1925–1957). Designed for strategic thinking, dialectical analysis, and organizational problem-solving.

## Core Frameworks

This skill provides five core frameworks distilled from Mao's work:

| Framework | Chinese | Key Concept | Quick Use |
|-----------|---------|-------------|-----------|
| Contradiction Analysis | 矛盾论 | Principal vs. secondary contradictions; internal vs. external causes | When facing complex, multi-factor problems |
| Practice Theory | 实践论 | Knowledge → Practice → Knowledge cycle; perceptual → rational cognition | When theory and reality diverge |
| Military Strategy | 军事战略 | Concentration of forces; active defense; guerrilla principles | Competitive strategy, resource allocation |
| Mass Line | 群众路线 | From the masses, to the masses; investigation over assumption | Product-market fit, user research |
| United Front | 统一战线 | Unite the many, isolate the few; independence within alliance | Partnership strategy, coalition building |

### When to Use Which Framework

1. **"I'm facing a messy problem with many conflicting factors"** → `frameworks/contradiction-analysis.md`
2. **"My plan isn't working in practice"** → `frameworks/practice-theory.md`
3. **"I'm outnumbered / under-resourced against a stronger opponent"** → `frameworks/military-strategy.md`
4. **"I need to understand what users/stakeholders really need"** → `frameworks/mass-line.md`
5. **"I need to build a coalition but maintain independence"** → `frameworks/united-front.md`

- **Cognitive methods**: Chapters 反对本本主义, 改造我们的学习, 实践论
- **Dialectics in action**: Chapters 论十大关系, 关于正确处理人民内部矛盾的问题

## Using This Skill

When invoked, I will:
1. Identify which framework(s) apply to your situation
2. Reference specific articles and passages

## ⚠️ Context Note

These writings span China's revolutionary period (1925–1957). The frameworks are extracted for their **analytical and strategic value**, abstracted from their specific historical context. Apply the method, not the historical conclusions.
