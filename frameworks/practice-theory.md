# Practice Theory · 实践论

> *"If you want to know the taste of a pear, you must change the pear by eating it yourself."*
> *"你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。"*

## The Knowledge Cycle · 认识循环

```
PRACTICE (实践)
    ↓
PERCEPTUAL KNOWLEDGE (感性认识) — raw experience, surface phenomena
    ↓
RATIONAL KNOWLEDGE (理性认识) — concepts, judgments, inferences
    ↓
PRACTICE (实践) — test, verify, refine
    ↓
(cycle repeats, each iteration deeper)
```

## Core Concepts

### 1. Practice as Source · 实践是认识的来源

All knowledge originates from practice. **You cannot understand something purely through reading or thinking about it.** Direct engagement is irreplaceable.

**Application:** Before forming strong opinions on a domain, get hands-on experience. "没有调查就没有发言权" (No investigation, no right to speak.)

### 2. Perceptual → Rational · 感性→理性

Raw experience (perceptual knowledge) must be processed into concepts and theories (rational knowledge). **Experience without theory is blind; theory without experience is empty.**

**Application:** After any significant experience, formalize it into principles. Write down: "What happened → What pattern I see → What rule this suggests."

### 3. Practice as Test · 实践是检验真理的标准

The only way to verify knowledge is through practice. **A theory that cannot be tested in practice is not useful knowledge.**

**Application:** Every strategy must have a feedback loop. Ship fast, measure, adjust.

### 4. Spiral Development · 螺旋式发展

Knowledge develops in cycles, each cycle reaching a higher level. **You never return to the same point; each iteration deepens understanding.**

## Decision-Making Flow

```
1. INVESTIGATE  → Collect facts, firsthand observation (调查)
2. STUDY        → Identify patterns, form hypotheses (研究)
3. DECIDE       → Make a concrete plan (决策)
4. ACT          → Execute (执行)
5. REFLECT      → Compare results with expectations (总结)
   ↓
   Return to Step 1 with deeper understanding
```

## Key Quotes

| Quote | Application |
|-------|-------------|
| "调查就像'十月怀胎'，解决问题就像'一朝分娩'" | Investigation is pregnancy; solution is birth. Don't rush to solutions. |
| "实践、认识、再实践、再认识" | Practice, knowledge, again practice, again knowledge — the cycle is infinite |
| "去粗取精、去伪存真、由此及彼、由表及里" | Discard the dross, select the essential; eliminate the false, retain the true; proceed from one to the other; from the outside to the inside |

## Anti-Patterns

- **Book worship · 本本主义/教条主义**: Treating texts as absolute truth without testing against reality
- **Empiricism · 经验主义**: Relying only on experience without extracting principles
- **Premature theorization**: Forming theories before sufficient investigation
- **Analysis paralysis**: Endless study without action — you learn more by doing
